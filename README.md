## 🏗️ Development Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Arnav-lunatic/laxmiSale.git
   cd laxmiSale
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create `.env` file:
   ```env
   VITE_CLOUDINARY_URL=your_cloudinary_url
   ```

4. **Start Development Server**
   ```bash
   npm run dev
   ```

5. **Build for Production**
   ```bash
   npm run build
   ```
