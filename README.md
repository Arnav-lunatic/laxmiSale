1. **Clone the Repository**
   ```bash
   git clone https://github.com/Arnav-lunatic/laxmiSale.git
   cd laxmiSale
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**

